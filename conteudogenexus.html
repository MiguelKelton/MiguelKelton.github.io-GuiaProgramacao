GeneXus

O que é

GeneXus é uma plataforma de desenvolvimento de software de baixo código (low-code) impulsionada por inteligência artificial que automatiza e simplifica a criação, evolução e manutenção de aplicações empresariais. Essencialmente, é um sistema que permite que desenvolvedores criem software de forma mais rápida e eficiente, utilizando representação de conhecimento e gerando código nativo para diversas plataformas, como web, dispositivos móveis e sistemas Windows.

Objetos

Transaction

O que é um Transaction?

Um Transaction representa uma entidade do mundo real, como por exemplo: Cliente, Produto ,Pedido, Funcionário
Quando você cria uma transação, você está dizendo ao GeneXus:
“Quero armazenar informações sobre X, com tais atributos, e com tais relações.”

O que ela faz?

Ao criar uma Transaction, GeneXus: Gera automaticamente a estrutura da tabela no banco de dados. Cria a interface de cadastro (tela) para inserir, editar e excluir dados. Define regras como chave primária, relacionamento e integridade referencial.
Não precisa escrever SQL nem montar telas manualmente. Você modela no Transaction e o GeneXus: Cria a tabela no banco, Cria os relacionamentos corretos, Gera a interface de cadastro automaticamente Aplica as regras de integridade

Dominio

O que é

Domínios em GeneXus são definições reutilizáveis de tipos de dados
Tipo de dado (por exemplo: Character, Numeric, Date, etc.)
Tamanho e formato
Regras de validação (opcional)
Máscara de exibição, descrição, etc.

Assim, em vez de configurar esses detalhes repetidamente para cada atributo, você os define uma vez no domínio e reutiliza onde quiser.

Como definir um dominio

Id=Numeric(4,0)
