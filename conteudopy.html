Python

O que é

Python é uma linguagem de programação de alto nível, conhecida por sua simplicidade e legibilidade. Criada por Guido van Rossum, ela possui uma sintaxe clara e intuitiva, tornando-se ideal para iniciantes e ao mesmo tempo poderosa para desenvolvedores experientes.
Ela é amplamente usada em diversas áreas, como desenvolvimento web, automação, ciência de dados, inteligência artificial, machine learning e segurança cibernética. Python é uma linguagem interpretada e multiplataforma, o que significa que pode ser executada em diferentes sistemas operacionais sem necessidade de compilação.
Com uma vasta biblioteca padrão e uma comunidade ativa, Python é uma das linguagens mais populares do mundo da programação.

Funções

If e Else:

Executa um bloco de código se a condição for verdadeira, senão executa outro bloco.

idade = 18

if idade >= 18:

print("Você é maior de idade.")

else:

print("Você é menor de idade.")

Elif

(Else If)

elif pode ser usada para verificar múltiplas condições.

nota = 7

if nota >= 9:

print("Excelente!")

elif nota >= 7:

print("Bom trabalho!")

elif nota >= 5:

print("Você passou!")

else:

print("Você reprovou.")
