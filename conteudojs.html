JavaScript

O que é

JavaScript é uma linguagem de programação amplamente utilizada para tornar páginas da web interativas. Ela permite criar efeitos dinâmicos, validar formulários, manipular elementos da página e até desenvolver aplicações completas no lado do cliente (frontend) e no lado do servidor (backend, com Node.js).
É uma linguagem interpretada, baseada em eventos e orientada a objetos, sendo um dos pilares do desenvolvimento web, junto com HTML e CSS. JavaScript também é muito usado no desenvolvimento de jogos, aplicativos móveis e sistemas de automação.

Usada para carregar um arquivo JavaScript externo em uma página HTML.

<script src="script.js"></script>

Variáveis

Uma variável serve para armazenar dados temporários em um programa, permitindo que você manipule, acesse e modifique esses dados

Const:

Const não pode ser redefinida.
têm escopo de bloco {} (só são acessíveis dentro do bloco onde foram declaradas).

const text = "texto";

Let:

O valor pode ser alterado, mas não redeclarado no mesmo escopo.
têm escopo de bloco {} (só são acessíveis dentro do bloco onde foram declaradas).

let text = "texto";

Var:

pode ser reatribuído e redeclarado dentro do mesmo escopo.
var tem escopo de função (visível em toda a função onde foi declarada).

var text ="texto"

Expressões e operadores

Os operadores são usados para manipular esses valores e realizar operações matemáticas, lógicas e de comparação.

Operações Matemáticas (Aritméticas)

let a = 10, b = 5;

console.log(a + b);

console.log(a - b);

console.log(a * b);

console.log(a / b);

console.log(a % b);

0 (resto da divisão)

console.log(2 ** 3);

8 (2 elevado a 3)

Operadores de Comparação

(10 > 5) Maior que

(10 < 5) Menor que

(5 >= 5) Maior ou igual

(7 <= 4) Menor ou igual

(5 == "5") // true (compara só o valor)

(5 === "5") // false (compara valor e tipo)

(5 != 3) Diferente

(5 !== "5") Diferente estrito



Operadores Lógicos

(5 > 3 && 10 > 5) // true (ambas são verdadeiras)

(5 > 3 || 10 < 5) // true (pelo menos uma é verdadeira)

(!(5 > 3)) // false (inverte o resultado)

Estruturas de controle de fluxo

If e Else:

Executa um bloco de código se a condição for verdadeira, senão executa outro bloco.

let idade = 18;

if (idade >= 18) {

console.log("Você é maior de idade.");

} else {

console.log("Você é menor de idade.");
}

If, Else If e Else:

Permite verificar várias condições antes de executar um else.

let nota = 75;
if (nota >= 90) {
console.log("Nota A");
} else if (nota >= 80) {
console.log("Nota B");
} else if (nota >= 70) {
} else if (nota >= 70) {
console.log("Nota C");
} else if (nota >= 70) {
} else {
console.log("Nota F");
}

Switch

(Estrutura de Escolha) Usado quando há várias possibilidades.

let dia = 3;

switch (dia) {
case 1:
console.log("Domingo");
break;
case 2:
console.log("Segunda-feira");
break;
case 3:
console.log("Terça-feira");
break;
default:
console.log("Dia inválido");
}

While

(Enquanto) Executa enquanto a condição for verdadeira.

let contador = 0;

while (contador < 5) {

console.log("Número:", contador);

contador++;

}

Do While

(Faça Enquanto) Executa pelo menos uma vez, depois verifica a condição.

let contador = 0;

do {

console.log("Número:", contador);

contador++;

} while (contador < 5);

For

(Para) Usado quando se sabe o número de repetições.

for (let i = 0; i < 5; i++) {

console.log("Número:", i);

}

Manipulação do DOM

A Manipulação do DOM (Document Object Model) no JavaScript permite alterar elementos HTML, modificar estilos e reagir a eventos da página.

Seleciona por ID

document.getElementById("meuElemento");

Seleciona a primeira classe encontrada

document.querySelector(".minhaClasse");

Seleciona todos os elementos com essa classe

document.querySelectorAll(".minhaClasse");

Seleciona por tag

document.getElementsByTagName("p");

Seleciona elementos pela classe

document.getElementsByClassName("minhaClasse");
