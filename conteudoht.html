HTML

O que é

HTML é uma linguagem de marcação utilizada para estruturar páginas web. Ele define os elementos e a organização do conteúdo.

Estrutura

A estrutura básica de um documento HTML é composta por várias tags e elementos que definem a página.

<!DOCTYPE html>

<html lang="pt-br">

<head>

<meta charset="UTF-8">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>Document</title>

</head>

<body>

</body>

</html>
